# 版本管理规范

## 版本命名规则

```
[方案代码]_[交付物]_v[主版本].[次版本].[修订号]_[日期]

示例：
XGCB_powerbank_front_v1.0.0_20260415.ai
XGCB_powerbank_back_v1.0.0_20260415.ai
XGCB_cabinet_front_v1.0.0_20260415.ai
GGXN_powerbank_front_v1.0.0_20260415.ai
```

## 方案代码

| 代码 | 方案名称 |
|------|---------|
| XGCB | 星轨相伴 |
| GGXN | 光轨逆行 |

## 交付物代码

| 代码 | 交付物名称 |
|------|---------|
| powerbank_front | 充电宝正面 |
| powerbank_back | 充电宝背面 |
| powerbank_all | 充电宝全套 |
| cabinet_front | 机柜正面 |
| cabinet_side | 机柜侧面 |
| cabinet_all | 机柜全套 |

## 版本更新规则

| 版本类型 | 更新内容 | 示例 |
|---------|---------|------|
| 主版本 | 设计方案重大变更 | v1.0.0 → v2.0.0 |
| 次版本 | 局部设计调整 | v1.0.0 → v1.1.0 |
| 修订号 | 细节修正、错误修复 | v1.0.0 → v1.0.1 |
