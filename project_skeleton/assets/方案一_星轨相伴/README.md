## 组合规则

### 正面组合

```
1. 放置渐变背景
2. 放置美团Logo（中心偏上）
3. 放置主星轨（环绕Logo）
4. 放置辅星轨（装饰）
5. 放置星点（分布，注意平衡）
6. 放置QR码（中心偏下）
7. 放置"此方向归还"（顶部）
```

### 背面组合

```
1. 放置渐变背景
2. 放置北斗七星连线（装饰）
3. 放置星点（分布）
4. 放置"此方向归还"（顶部）
5. 放置条形码（底部）
```

- 颜色：黄绿色
- 工艺：丝印叠加
