## 组合规则

### 正面组合

```
1. 放置美团黄底色
2. 放置主光轨（5条，从底部向上流动）
3. 放置辅光轨（8条，装饰性）
4. 放置城市轮廓（光轨终点）
5. 放置美团Logo（中心偏上，被光轨环绕）
6. 放置QR码（左下角）
7. 放置"此方向归还"（右下角）
```

### 背面组合

```
1. 放置美团黄底色
2. 放置光轨汇聚效果（向三线出口汇聚）
3. 放置能量发射口（三线出口区域）
4. 放置"此方向归还"（顶部）
5. 放置条形码（底部）
```

### 机柜组合

```
1. 放置美团黄底色
2. 放置光轨网络（连接各站点）
3. 放置12个站点图标（2列×6行）
4. 放置美团Logo（头部左侧）
5. 放置"美团·充电"（头部中央）
6. 放置QR码（头部右侧）
```

## 城市延展说明

- 每个城市版本替换城市轮廓素材
- 光轨方向和数量保持一致
- 可根据城市特色调整配色（如成都可增加熊猫元素）
