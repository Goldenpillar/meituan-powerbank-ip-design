# 美团充电宝「星轨相伴」设计细节文档

---

## 一、设计概述

### 1.1 方案主题
**「点亮生活，随行有你」**

### 1.2 核心创意
- **夜光星空**：充电宝在暗处自动发光，营造梦幻星空效果
- **动态星轨**：以美团Logo为中心，星轨环绕旋转的视觉动势
- **情感连接**：星空象征陪伴与指引，与充电宝"能量补给"的属性高度契合

### 1.3 创新亮点
- **行业首创**：夜光油墨首次应用于共享充电宝领域
- **差异化记忆点**：暗处发光特性形成强品牌识别
- **社交传播价值**：用户自发拍照分享，形成二次传播

## 五、工艺说明

### 5.1 充电宝工艺

#### 【印刷工艺组合】

| 工艺 | 应用区域 | 说明 |
|------|----------|------|
| UV彩印 | 整体渐变背景 | 高精度数码印刷，色彩还原度高 |
| 丝印 | 美团Logo、文字 | 白色油墨，高遮盖力 |
| 夜光油墨 | 星点、星轨、流星头部 | 储光型夜光油墨，发光时长4-6小时 |
| UV上光 | Logo区域、QR码区域 | 局部高光效果，增强质感 |

#### 【夜光油墨技术参数】
| 项目 | 规格 |
|------|------|
| 类型 | 储光型夜光油墨（稀土铝酸盐系） |
| 发光颜色 | 黄绿色（波长520nm） |
| 激发时间 | 10-15分钟（室内光照） |
| 发光时长 | 4-6小时（亮度衰减至初始10%） |
| 印刷厚度 | 30-50μm |
| 耐磨性 | ≥500次摩擦测试 |

#### 【工艺流程】
1. **底漆喷涂**：白色底漆，增强色彩附着力
2. **UV彩印**：渐变背景一次成型
3. **丝印白色**：Logo、文字、QR码背景
4. **丝印夜光**：星点、星轨元素
5. **UV上光**：局部高光处理
6. **保护清漆**：全表面哑光保护层

### 5.2 机柜工艺

#### 【印刷工艺组合】

| 工艺 | 应用区域 | 说明 |
|------|----------|------|
| 粉末喷涂 | 机柜外壳 | 星云渐变色，耐候性强 |
| UV印刷 | 面板装饰 | 星轨、星点图案 |
| 亚克力丝印 | Logo面板 | 品牌标识 |
| LED背光 | Logo区域 | 内置LED模组 |

#### 【LED模组参数】
| 项目 | 规格 |
|------|------|
| 类型 | SMD LED模组 |
| 颜色 | 暖白光（3000K） |
| 功率 | 0.5W/模组 |
| 数量 | 6个（Logo背光） |
| 寿命 | ≥50000小时 |

### 5.3 工艺落地可行性分析

#### 【技术可行性】
| 项目 | 评估 | 说明 |
|------|------|------|
| UV彩印渐变 | ★★★★★ | 成熟工艺，色彩过渡自然 |
| 夜光油墨 | ★★★★☆ | 首次应用于充电宝，需打样验证 |
| LED背光 | ★★★★★ | 常规工艺，稳定可靠 |
| 粉末喷涂渐变 | ★★★★☆ | 需精确控制喷涂区域 |

#### 【生产可行性】
- **充电宝**：现有生产线可兼容，需增加夜光油墨丝印工位
- **机柜**：需调整喷涂工艺，增加LED背光组装工序
- **产能影响**：预计增加15-20%生产周期
