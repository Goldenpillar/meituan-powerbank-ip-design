# 星轨相伴 — 美团充电宝IP涂装设计工作区

## 项目概述

- **方案名称**：星轨相伴（Xing Gui Xiang Ban）
- **方案代号**：XGCB
- **项目类型**：美团充电宝IP涂装设计
- **设计主题**：以星空、星轨、星云为核心视觉元素，将充电宝的使用体验隐喻为"在星夜中相伴同行"的浪漫意象。渐变色彩从美团品牌黄过渡到深邃星空蓝，结合夜光油墨工艺，在暗光环境下呈现独特的星空效果。

## 设计理念

**星轨相伴** — 每一次充电，都是一段星夜旅程的陪伴。

- **色彩**：从美团品牌黄（#FFC600）渐变至深空蓝（#1A1A4E），象征从日间到夜幕的自然过渡
- **星轨**：环绕美团Logo的弧形线条，象征充电宝与用户之间的紧密连接
- **星点**：大小不一的星点散布于机身，采用夜光油墨工艺，暗光环境下呈现梦幻效果
- **星云**：机柜采用紫蓝色星云渐变，营造沉浸式星空氛围

## 工艺说明

- **UV彩印**：主体图案采用UV彩色印刷，色彩饱和度高
- **夜光油墨**：星点区域采用夜光油墨，吸收光线后在暗环境中发出柔和光芒
- **丝印**：品牌标识和功能提示采用丝印工艺，确保耐久性

## 注意事项

1. 所有印刷尺寸文件必须以300dpi输出，色彩模式为CMYK
2. 预览文件使用RGB色彩模式
3. 品牌元素（Logo、QR码、归还提示）的位置和尺寸不得随意更改
4. 夜光油墨区域需在设计稿中明确标注
5. 三线出口区域为功能区域，设计时需预留足够空间
