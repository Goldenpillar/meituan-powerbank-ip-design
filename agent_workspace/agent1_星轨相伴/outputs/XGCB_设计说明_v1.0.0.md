## 一、设计理念

### 1.1 核心创意

「星轨相伴」将充电宝隐喻为"个人星系中的能量星体"——每个用户都是自己生活宇宙的中心，而充电宝则是那颗默默提供能量的恒星。

**三大核心意象**：

| 意象 | 含义 | 设计表达 |
|------|------|---------|
| 星轨 | 能量围绕品牌核心运转 | 3条椭圆弧线环绕美团Logo |
| 星点 | 生命中的闪光时刻 | 45颗大小不一的夜光星点 |
| 星云 | 无限可能的宇宙 | 机柜紫蓝渐变星云效果 |

### 1.2 情感诉求

- **陪伴感**：星空象征长久的陪伴，与充电宝"随时随地为手机续航"的产品属性高度契合
- **浪漫感**：夜光效果在暗光环境中呈现梦幻星空，创造独特的情感记忆点
- **品质感**：黄蓝渐变配色兼具品牌识别度与高级感，区别于传统纯色充电宝

### 1.3 创新亮点

- **行业首创**：夜光油墨首次应用于共享充电宝领域，暗处星点可持续发光4-6小时
- **差异化记忆点**：在酒吧、影院、夜店等暗光场景中，充电宝自动成为视觉焦点
- **社交传播价值**：用户自发拍照分享夜光效果，形成二次传播

```

   整体为主        底部微带紫色
```

```
#2D1B4E (深紫) → #4A3C8E (紫蓝) → #0A0A2E (深蓝)
   0-80mm          80-240mm          240-360mm
```

## 三、工艺说明

### 3.1 充电宝工艺

| 工艺类型 | 应用区域 | 具体说明 |
|----------|----------|---------|
| **UV彩印** | 渐变背景、星轨线条、小星点、流星拖尾 | 高精度数码印刷，色彩还原度>95%，分辨率1200dpi |
| **夜光油墨** | 大星点(5颗)、中星点(12颗)、星轨弧线(3条)、流星头部(2个) | 储光型稀土铝酸盐系，激发10-15分钟，发光4-6小时，印刷厚度30-50um |
| **丝印** | 美团Logo、"此方向归还"文字、QR码 | 白色油墨，高遮盖力，耐磨>500次摩擦 |
| **UV上光** | Logo区域、QR码区域 | 局部高光效果，增强质感和耐久性 |
| **保护清漆** | 全表面 | 哑光保护层，防刮防指纹 |

### 3.2 机柜工艺

| 工艺类型 | 应用区域 | 具体说明 |
|----------|----------|---------|
| **粉末喷涂** | 机柜外壳 | 星云渐变色，耐候性强，附着力>3M胶带测试 |
| **UV印刷** | 面板装饰（星轨、星点、星座连线） | 高精度数码印刷，户外耐候级油墨 |
| **亚克力丝印** | Logo面板、品牌文字 | 品牌标识区域 |
| **LED背光** | Logo区域、充电仓指示灯 | SMD LED模组，暖白光3000K，寿命>50000小时 |
| **夜光油墨** | 充电仓星点造型(12个)、星团效果(4个) | 与充电宝统一夜光效果 |

### 3.3 工艺流程

**充电宝**：
1. 底漆喷涂（白色底漆，增强附着力）
2. UV彩印（渐变背景一次成型）
3. 丝印白色（Logo、文字、QR码背景）
4. 丝印夜光（星点、星轨、流星头部）
5. UV上光（局部高光处理）
6. 保护清漆（全表面哑光保护层）

**机柜**：
1. 钣金外壳成型
2. 粉末喷涂（星云渐变色）
3. 亚克力面板安装
4. UV印刷装饰图案
5. 丝印品牌标识
6. LED背光模组组装

## 五、与标准版差异对比

| 对比维度 | 标准版 | 星轨相伴版 | 差异说明 |
|----------|--------|-----------|---------|
| **主色调** | 美团黄 #FFC600 纯色 | 黄蓝渐变 #FFC600→#1A1A4E | 从单色升级为渐变，增加视觉层次 |
| **背景纹理** | 纯色无纹理 | 星空纹理+星云效果 | 增加深度感和品质感 |
| **装饰元素** | 无 | 星轨+星点+流星+星座连线 | 核心创意差异化 |
| **特殊工艺** | 普通UV彩印 | UV彩印+夜光油墨 | 新增夜光功能，暗光场景差异化 |
| **Logo处理** | 黑色圆角矩形内白字 | 白色文字+星轨环绕 | Logo视觉权重提升 |
| **QR码处理** | 黑色QR码 | 白底圆角矩形+中心Logo | 提升扫码成功率和品牌一致性 |
| **机柜主题** | 纯美团黄 | 紫蓝星云渐变 | 营造沉浸式星空氛围 |
| **充电仓设计** | 标准椭圆凹槽 | 星点造型+LED发光边框 | 功能性与美观性兼顾 |
| **情感价值** | 功能导向 | 陪伴+浪漫+品质 | 提升品牌情感连接 |
| **社交属性** | 低（纯功能产品） | 高（夜光效果可拍照分享） | 增加自然传播能力 |
| **单体成本增量** | 基准 | +1.6~2.5元/台 | 夜光油墨+UV彩印增量 |
| **目标场景** | 通用场景 | 全场景+暗光场景强化 | 酒吧/影院/夜店等场景优势明显 |
