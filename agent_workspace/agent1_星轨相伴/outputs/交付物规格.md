## 一、充电宝正面（powerbank/）

### 1.1 基本信息

| 属性 | 规格 |
|------|------|
| **文件名** | XGCB_powerbank_front_v1.0.0.png |
| **存放路径** | outputs/powerbank/ |
| **印刷尺寸** | 143mm x 71mm |
| **输出分辨率** | 300dpi |
| **像素尺寸** | 1693px x 839px（300dpi换算） |
| **色彩模式** | CMYK（印刷用）/ RGB（预览用） |
| **文件格式** | PNG（最终交付）/ AI/PSD（源文件） |

### 1.2 必须包含的品牌元素

以下品牌元素为**强制要求**，不可省略、遮挡或偏移：

| 元素 | 位置 | 尺寸 | 说明 |
|------|------|------|------|
| 美团Logo | 居中偏上 | 按品牌规范 | 丝印工艺，确保清晰可辨 |
| QR码 | 下方区域 | 22mm x 22mm | 丝印工艺，需添加白色底衬确保扫描识别 |
| "此方向归还"提示 | 底部 | 按品牌规范 | 丝印工艺，方向箭头需清晰 |

### 1.3 设计元素

| 元素 | 说明 | 对应素材 |
|------|------|----------|
| 渐变背景 | 从顶部美团黄（#FFC600）渐变到底部深蓝（#1A1A4E），中间经过橙色和紫色的过渡 | gradient_front.png |
| 主星轨 | 环绕美团Logo的弧形线条，宽度2-3mm，白色/浅金色 | orbit_main.png |
| 辅星轨 | 与主星轨配合的弧形线条，宽度1-1.5mm，白色/浅蓝色 | orbit_secondary.png |
| 大星点 | 3-5个，直径8mm，分布于渐变背景的深色区域 | star_large.png |
| 中星点 | 5-8个，直径5mm，填充星空密度 | star_medium.png |
| 小星点 | 10-15个，直径2mm，增加层次感 | star_small.png |

### 1.4 工艺标注要求

设计稿中必须包含以下工艺标注图层：

| 工艺类型 | 标注区域 | 标注颜色建议 | 说明 |
|----------|----------|-------------|------|
| UV彩印 | 渐变背景、星轨线条、小星点 | 专色标注 | 主体图案印刷区域 |
| 夜光油墨 | 大星点、中星点 | Pantone 803C或指定夜光色 | 吸收光线后在暗环境中发光 |
| 丝印 | 美团Logo、QR码、"此方向归还"提示 | 单色标注 | 品牌标识和功能提示 |

### 1.5 设计注意事项

1. 品牌元素（Logo、QR码、归还提示）的位置需与标准版参照图保持一致
2. 星轨线条不可遮挡品牌元素
3. 夜光油墨区域需在独立图层中标注，便于印刷厂识别
4. 渐变过渡需自然平滑，不可出现明显的色带或断层
5. 安全区域内缩3mm，所有设计元素不可超出安全区域

## 二、充电宝背面（powerbank/）

### 2.1 基本信息

| 属性 | 规格 |
|------|------|
| **文件名** | XGCB_powerbank_back_v1.0.0.png |
| **存放路径** | outputs/powerbank/ |
| **印刷尺寸** | 143mm x 71mm |
| **输出分辨率** | 300dpi |
| **像素尺寸** | 1693px x 839px（300dpi换算） |
| **色彩模式** | CMYK（印刷用）/ RGB（预览用） |
| **文件格式** | PNG（最终交付）/ AI/PSD（源文件） |

### 2.2 必须包含的品牌元素

| 元素 | 位置 | 尺寸 | 说明 |
|------|------|------|------|
| "此方向归还"提示 | 底部 | 按品牌规范 | 丝印工艺，方向箭头需清晰 |
| 条形码 | 指定位置 | 35mm x 12mm | 丝印工艺，需添加白色底衬确保扫描识别 |
| 三线出口区域 | 按模板标注 | — | 绝对禁止设计区域，不可放置任何装饰元素 |

### 2.3 设计元素

| 元素 | 说明 | 对应素材 |
|------|------|----------|
| 渐变背景 | 以深蓝（#1A1A4E）为主，可加入微弱的紫色渐变 | gradient_back.png |
| 彗星尾巴效果 | 1-2条彗星尾巴，从一角划过，营造动态感 | orbit_comet.png |
| 星点装饰 | 少量小星点和中星点，与正面呼应但密度较低 | star_small.png、star_medium.png |
| 星座连线（可选） | 若空间允许，可加入简洁的星座连线图案 | constellations/ |

### 2.4 工艺标注要求

| 工艺类型 | 标注区域 | 标注颜色建议 | 说明 |
|----------|----------|-------------|------|
| UV彩印 | 渐变背景、彗星尾巴、小星点 | 专色标注 | 主体图案印刷区域 |
| 夜光油墨 | 中星点（如有） | Pantone 803C或指定夜光色 | 与正面夜光效果呼应 |
| 丝印 | "此方向归还"提示、条形码 | 单色标注 | 功能提示和追溯码 |

### 2.5 设计注意事项

1. 三线出口区域（USB-C和Lightning接口）为**绝对禁止设计区域**
2. 彗星尾巴的尾部方向应朝向充电宝边缘，营造"划过"的动感
3. 背面整体色调偏深，确保与正面的黄蓝渐变形成"日与夜"的对比
4. 条形码区域需保证足够的对比度，添加白色底衬

## 三、机柜正面（cabinet/）

### 3.1 基本信息

| 属性 | 规格 |
|------|------|
| **文件名** | XGCB_cabinet_front_v1.0.0.png |
| **存放路径** | outputs/cabinet/ |
| **印刷尺寸** | 250mm x 360mm（近似值，以实际机柜尺寸为准） |
| **输出分辨率** | 300dpi |
| **像素尺寸** | 2953px x 4252px（300dpi换算） |
| **色彩模式** | CMYK（印刷用）/ RGB（预览用） |
| **文件格式** | PNG（最终交付）/ AI/PSD（源文件） |

### 3.2 必须包含的品牌元素

| 元素 | 位置 | 尺寸 | 说明 |
|------|------|------|------|
| 美团Logo | 头部品牌区域 | 按品牌规范 | 丝印工艺，需在星云背景上保证可读性 |
| "美团·充电"文字 | 头部品牌区域 | 按品牌规范 | 丝印工艺 |
| QR码 | 头部或侧面 | 22mm x 22mm | 丝印工艺，需添加白色底衬 |
| 12个充电仓 | 2列 x 6行布局 | 按模板标注 | 充电仓开口不可被设计元素遮挡 |

### 3.3 设计元素

| 元素 | 说明 | 对应素材 |
|------|------|----------|
| 星云渐变背景 | 紫蓝色星云效果，覆盖整个面板，多层渐变叠加 | gradient_cabinet.png |
| 充电仓星点设计 | 12个充电仓的边框或指示灯区域设计成"星点"造型 | star_large.png（参考） |
| 星轨线条 | 从头部品牌区域延伸至充电仓区域的弧形线条 | orbit_main.png、orbit_secondary.png |
| 星团效果 | 2-4个星团，分布在面板的不同区域 | star_cluster.png |
| 星点分布 | 大量大小不一的星点，营造深邃星空氛围 | star_large/medium/small.png |

### 3.4 工艺标注要求

| 工艺类型 | 标注区域 | 标注颜色建议 | 说明 |
|----------|----------|-------------|------|
| UV彩印 | 星云渐变背景、星轨线条、星点装饰 | 专色标注 | 主体图案印刷区域 |
| 夜光油墨 | 星团效果、充电仓星点设计 | Pantone 803C或指定夜光色 | 暗环境中充电仓位置一目了然 |
| 丝印 | 美团Logo、"美团·充电"文字、QR码 | 单色标注 | 品牌标识 |

### 3.5 设计注意事项

1. 头部品牌区域需保证Logo和文字的清晰可读性，必要时在Logo下方添加半透明深色底衬
2. 充电仓开口为功能区域，设计元素不可遮挡仓口
3. 星云渐变需有层次感，避免单调的纯色渐变
4. 充电仓的"星点"设计需兼顾美观和功能性，不影响用户识别仓口位置
5. 夜光油墨在机柜上的应用可更加大胆，因为机柜通常放置在光线较弱的室内环境

## 四、交付物检查清单

### 4.1 文件完整性检查

- [ ] 文件名符合版本命名规则（XGCB_[交付物]_v[主版本].[次版本].[修订号]）
- [ ] 文件格式正确（PNG用于最终交付，AI/PSD用于源文件）
- [ ] 文件尺寸和分辨率符合规格要求
- [ ] 色彩模式正确（CMYK用于印刷，RGB用于预览）

### 4.2 品牌元素检查

- [ ] 美团Logo位置和比例正确
- [ ] QR码完整且可扫描
- [ ] "此方向归还"提示位置正确
- [ ] 条形码完整且可扫描（背面）
- [ ] "美团·充电"文字完整（机柜）

### 4.3 设计元素检查

- [ ] 渐变背景过渡自然平滑
- [ ] 星轨线条环绕Logo，不遮挡品牌元素
- [ ] 星点分布均匀，视觉平衡
- [ ] 夜光油墨区域已标注
- [ ] 三线出口区域无设计元素（背面）

### 4.4 工艺标注检查

- [ ] UV彩印区域已标注
- [ ] 夜光油墨区域已标注（使用专色）
- [ ] 丝印区域已标注
- [ ] 标注图层独立，便于印刷厂识别
