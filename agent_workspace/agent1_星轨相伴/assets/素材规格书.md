### 2.1 orbit_main.png — 主星轨

| 属性 | 规格 |

| **文件名** | orbit_main.png |

| **格式** | PNG（透明背景） |
| **色彩** | 白色/浅金色，带轻微渐变（从亮到暗） |
| **分辨率** | 300dpi（印刷级） |

## 七、素材使用规范

### 7.1 通用规范

1. 所有素材文件使用透明背景（PNG格式），便于叠加拼装
2. 印刷级素材分辨率不低于300dpi
3. 素材文件的色彩模式需与最终输出一致（CMYK用于印刷，RGB用于预览）
4. 素材文件命名需严格遵循本文档定义的命名规则

### 7.2 拼装流程

1. **底色层**：放置渐变背景素材（gradient_*.png）
2. **轨道层**：放置星轨素材（orbit_*.png），调整位置和角度
3. **星点层**：放置星点素材（star_*.png），随机分布但保持视觉平衡
4. **品牌层**：放置品牌素材（meituan_logo.ai、meituan_qrcode.png等）
5. **工艺标注层**：标注UV彩印区域、夜光油墨区域、丝印区域

### 7.3 夜光油墨标注规范

需要标注夜光油墨区域的素材：
- star_large.png — 整体标注
- star_medium.png — 整体标注
- star_cluster.png — 整体标注

标注方式：在设计稿中创建独立的"夜光油墨"图层，使用专色（建议Pantone 803C或指定夜光色）填充对应区域。
