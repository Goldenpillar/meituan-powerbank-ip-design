# 素材库规格说明

本目录为「光轨逆行」方案的可拼装素材库。所有素材需按照以下规格制作，并放入对应的子文件夹中。素材均为透明背景 PNG 或矢量 AI 格式，便于在最终设计中灵活拼装和调整。

---

## 一、光轨素材（light_rails/）

光轨是本方案的核心视觉元素，象征能量在充电网络中的流动。所有光轨素材需具备发光质感，线条边缘需有柔和的光晕效果。

### 1.1 rail_main.png — 主光轨线条

| 属性 | 规格 |
|------|------|
| 文件名 | rail_main.png |
| 格式 | PNG（透明背景） |
| 线条宽度 | 3-4mm |
| 方向 | 从底部向上流动的弧线 |
| 长度 | 60-80mm（可裁切使用） |
| 色彩 | 深色渐变（#1A1A2E → #0F3460），边缘带亮色光晕（#E94560） |
| 分辨率 | 300dpi |
| 说明 | 主视觉线条，用于充电宝正面和机柜正面，从底部向上弧形流动，表达能量逆重力传输 |

### 1.2 rail_secondary.png — 辅光轨线条

| 属性 | 规格 |
|------|------|
| 文件名 | rail_secondary.png |

| 方向 | 装饰性流动线条，与主光轨平行或交叉 |

## 素材使用注意事项

1. **透明背景**：所有 PNG 素材必须为透明背景，便于在设计中灵活拼装
2. **分辨率统一**：所有位图素材统一为 300dpi，确保印刷质量
3. **色彩一致性**：所有素材的色彩需参照 README.md 中的色彩体系，保持视觉统一
4. **矢量优先**：线条类素材建议同时提供 AI 矢量版本，便于放大缩小不失真
5. **命名规范**：素材文件名需与本文档中列出的文件名完全一致
