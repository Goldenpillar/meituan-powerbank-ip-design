# 交付物规格

本目录为「光轨逆行」方案的最终交付物存放位置。所有交付物需严格按照以下规格执行。

---

## 一、充电宝正面设计（outputs/powerbank/）

### 1.1 基本信息

| 属性 | 规格 |
|------|------|
| 文件名 | GGXN_powerbank_front_v1.0.0.png |
| 存放路径 | outputs/powerbank/ |
| 印刷尺寸 | 143mm x 71mm |
| 输出分辨率 | 300dpi |
| 像素尺寸 | 1693 x 839px（300dpi 时） |
| 色彩模式 | CMYK（印刷文件）/ RGB（预览文件） |
| 文件格式 | PNG（最终印刷转 TIFF 或 PDF） |

### 1.2 必须包含的元素

| 序号 | 元素 | 位置要求 | 说明 |
|------|------|----------|------|
| 1 | 美团 Logo | 正面中上部 | 使用品牌素材库中的官方矢量 Logo，不可修改 |
| 2 | QR 码 | 正面下部 | 使用品牌素材库中的 QR 码模板，尺寸 22x22mm |
| 3 | "此方向归还"提示 | QR 码下方或侧面 | 使用品牌素材库中的提示模板，保持标准样式 |

### 1.3 设计元素说明

| 序号 | 设计元素 | 描述 |
|------|----------|------|
| 1 | 底色 | 美团黄（#FFC600）作为整体底色，覆盖充电宝正面全部区域 |
| 2 | 主光轨 | 深色光轨线条（#1A1A2E → #0F3460）从底部向上弧形流动，宽度 3-4mm，2-3 条主光轨 |
| 3 | 辅光轨 | 亮色光轨线条（#E94560 → #FFC600）与主光轨平行或交叉，宽度 1.5-2mm，低透明度 |
| 4 | 城市轮廓 | 深蓝灰色（#2C2C54）城市天际线剪影位于底部，高度 30-40mm |
| 5 | Logo 光轨环绕 | 光轨线条在美团 Logo 周围形成环绕效果，突出品牌标识 |
| 6 | 汇聚点 | 光轨线条在 QR 码区域附近形成汇聚效果，引导用户扫码 |

### 1.4 工艺标注

| 工艺类型 | 区域 | 说明 |
|----------|------|------|
| UV 彩印 | 光轨线条区域 | 光轨线条使用 UV 彩印工艺，增强发光质感和立体感 |
| 丝印 | 美团 Logo、"此方向归还"文字 | 品牌文字使用丝印工艺，确保清晰度和耐久性 |

### 1.5 安全区域

- 裁切线外延 2mm（出血区域）
- 安全区域内缩 3mm（所有设计元素不得超出安全区域）
- 三线出口区域在背面，正面无特殊避让区域

| 序号 | 交付物 | 文件名 | 路径 | 格式 |
|------|--------|--------|------|------|
| 1 | 充电宝正面设计 | GGXN_powerbank_front_v1.0.0.png | outputs/powerbank/ | PNG / CMYK+RGB |

| 3 | 机柜正面设计 | GGXN_cabinet_front_v1.0.0.png | outputs/cabinet/ | PNG / CMYK+RGB |

### 交付文件要求

1. **双版本输出**：每个交付物需同时提供 CMYK（印刷用）和 RGB（屏幕预览用）两个版本
2. **源文件归档**：设计源文件（.ai 或 .psd）需一并归档至 versions/ 目录
3. **工艺说明文件**：每个交付物需附带工艺说明，标注 UV 彩印区域、丝印区域等
4. **色彩校验**：CMYK 文件需在标准印刷色域内，避免超出色域导致的颜色偏差

## 五、质量检查清单

交付前请逐项检查：

- [ ] 文件尺寸是否符合规格要求

- [ ] 美团 Logo 是否使用官方版本，未被修改

- [ ] 安全区域是否满足（内缩 3mm）
- [ ] 三线出口区域是否未被覆盖（背面）
- [ ] 充电仓区域是否未被遮挡（机柜）
- [ ] 光轨线条是否从底部向上流动（正面）
- [ ] 光轨是否向三线出口汇聚（背面）
- [ ] 城市轮廓是否包含且位置正确
- [ ] 工艺标注是否完整
- [ ] 文件命名是否符合版本管理规则
