# 美团充电宝IP宝涂装设计 —「光轨逆行」工作区

## 项目概述

- **项目名称**：美团充电宝IP宝涂装设计
- **方案名称**：光轨逆行（GGXN）
- **方案编号**：方案二
- **核心概念**：以「光轨」为视觉主线，深色光轨线条从充电宝底部向上流动，象征能量逆重力传输；城市轮廓剪影作为场景化元素，体现美团充电宝覆盖全国城市的网络化布局。

## 快速开始指南

### 第一步：了解参照资料

打开 `references/参照指南.md`，查看所有参照图片的说明和竞品避雷清单。参照图片位于 `references/` 目录下，可直接打开查看。

### 第二步：准备素材

打开 `assets/素材规格书.md`，了解需要制作的所有素材及其规格要求。按照目录结构将素材文件放入对应子文件夹中。

### 第三步：确认交付物要求

打开 `outputs/交付物规格.md`，了解最终交付物的尺寸、格式、色彩模式和必须包含的元素。

### 第四步：版本管理

打开 `versions/版本管理.md`，了解版本命名规则，确保文件命名规范统一。

## 设计核心要素

### 色彩体系

| 角色 | 色值 | 用途 |
|------|------|------|
| 品牌底色 | #FFC600（美团黄） | 充电宝/机柜底色 |
| 光轨主色 | 深色渐变（#1A1A2E → #0F3460） | 主光轨线条 |
| 光轨辅色 | 亮色渐变（#E94560 → #FFC600） | 辅光轨、发光效果 |
| 城市轮廓 | #2C2C54（深蓝灰） | 城市天际线剪影 |

### 设计关键词

- **光轨逆行**：能量从底部向上流动，打破重力方向
- **城市网络**：光轨连接城市站点，体现网络化覆盖
- **能量汇聚**：光轨向充电出口汇聚，体现充电功能
- **科技流动**：动态光轨线条营造科技感与速度感
