# 美团充电宝IP宝涂装设计

> 美团本地生活设计大赛 | 命题一：IP宝涂装设计
> 
> 主题：**点亮生活，随行有你**

## 项目简介

围绕美团充电宝产品形态，进行充电宝表面涂装及定制机柜视觉设计。参赛作品包含充电宝机身涂装设计（正面、背面），以及延展至配套定制机柜外观设计，形成完整的终端视觉方案。

## 方案概览

| 方案 | 名称 | 核心创新 | 主色调 |
|------|------|---------|--------|
| 方案一 | 星轨相伴 | 夜光油墨首次应用于共享充电宝 | 渐变（美团黄→深蓝） |
| 方案二 | 光轨逆行 | 光绘摄影视觉语言首次应用于充电宝 | 美团黄主底色 |

## 设计约束

- **落地可实施性**为第一优先级
- 需符合充电宝实际尺寸规格（143×71×16mm）及印刷工艺要求
- 须保留美团Logo、QR码、"此方向归还"等必要品牌元素
- 美团品牌色：#FFC600
