## 方案一「星轨相伴」素材库

### 基础素材

| 素材名称 | 格式 | 尺寸/规格 | 说明 |
|---------|------|----------|------|
| 星点-大 | PNG/AI | 直径5-8mm | 夜光油墨区域，可独立使用 |
| 星点-中 | PNG/AI | 直径3-5mm | 夜光油墨区域，可独立使用 |
| 星点-小 | PNG/AI | 直径1-2mm | 装饰性星点，可批量复制 |
| 星轨-主轨 | AI | 宽度2-3mm | 环绕Logo的主星轨 |
| 星轨-辅轨 | AI | 宽度1-1.5mm | 装饰性辅星轨 |
| 流星元素 | PNG/AI | 长度15-25mm | 动态感装饰元素 |
| 北斗七星连线 | AI | 可缩放 | 背面装饰元素 |
| 渐变背景-黄蓝 | AI | 143×71mm | 正面底色渐变 |

### 可拼装组合规则

```
组合规则：
1. 星点可自由分布，但需保持视觉平衡
2. 星轨必须环绕美团Logo，形成"轨道"效果
3. 流星元素建议放置于右上角或左下角，形成对角线动感
4. 夜光油墨仅应用于星点区域，星轨使用普通油墨
5. 北斗七星连线仅用于背面装饰

禁止规则：
1. 禁止星点遮挡QR码、Logo、归还提示
2. 禁止星轨穿过Logo中心
3. 禁止使用非品牌色系的颜色
```

| 城市轮廓-上海 | AI | 高度30-40mm | 含东方明珠、上海中心轮廓 |
| 城市轮廓-广州 | AI | 高度30-40mm | 含广州塔轮廓 |
| 能量发射口 | AI | 30×15mm | 三线出口区域专用 |
| 站点图标 | AI | 直径8mm | 机柜充电仓标识 |

### 可拼装组合规则

```
组合规则：
1. 光轨从底部向上流动，形成"上升"的视觉趋势
2. 主轨5条 + 辅轨8条，形成层次感
3. 城市轮廓位于光轨"终点"，形成天际线效果
4. 光轨环绕美团Logo，但不遮挡
5. 机柜站点以光轨网络连接

禁止规则：
1. 禁止光轨线条过于密集，影响QR码扫描
2. 禁止城市轮廓过于具象，保持抽象风格
3. 禁止使用非品牌色系的颜色
```

## 文件输出规范

### 文件格式

| 用途 | 格式 | 色彩模式 | 分辨率 |
|------|------|---------|--------|
| 设计源文件 | AI | CMYK | 矢量 |
| 印刷文件 | PDF | CMYK | 矢量 |
| 预览图 | PNG | RGB | 300dpi |
| 展示图 | JPG | RGB | 300dpi |

### 文件命名规范

```
[方案代码]_[交付物]_[版本]_[日期].[扩展名]

示例：
XGCB_powerbank_front_v1.0.0_20260415.ai
XGCB_powerbank_front_v1.0.0_20260415.pdf
GGXN_cabinet_front_v1.0.0_20260415.png
```

### 图层命名规范

```
图层结构：
├── 01_必要元素
│   ├── Logo
│   ├── QR码
│   ├── 归还提示
│   └── 条形码
├── 02_主视觉
│   ├── 背景
│   ├── 主元素
│   └── 装饰元素
├── 03_文字
│   ├── 品牌名
│   └── 标语
└── 04_标注（仅设计文件）
    ├── 尺寸标注
    ├── 工艺标注
    └── 色值标注
```

## 质量检查工具

### 设计检查清单

| 检查项 | 检查方法 | 通过标准 |
|--------|---------|---------|
| 尺寸准确性 | AI测量工具 | 误差<0.5mm |
| 色值正确性 | 吸管工具 | 符合品牌规范 |
| 元素完整性 | 图层检查 | 必要元素齐全 |
| 文字可读性 | 100%视图检查 | 清晰可读 |
| 工艺可行性 | 供应商确认 | 供应商确认可行 |

### 印刷检查清单

| 检查项 | 检查方法 | 通过标准 |
|--------|---------|---------|
| 色彩还原 | 色差仪 | ΔE<3 |
| 线条清晰度 | 放大镜检查 | 无锯齿、无断线 |
| 夜光效果 | 暗室测试 | 发光时长>4小时 |
| 附着力 | 胶带测试 | 无脱落 |
| 耐磨性 | 摩擦测试 | 无明显磨损 |
